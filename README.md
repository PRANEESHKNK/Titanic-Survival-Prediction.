# Titanic Survival Prediction

This project predicts passenger survival on the Titanic dataset using Machine Learning techniques.

## Tools and Technologies

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## Workflow

1. Data Cleaning
2. Handling Missing Values
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Model Evaluation

## Models Used

* Logistic Regression
* Decision Tree Classifier

## Results

The Logistic Regression model achieved approximately 78% accuracy on the test dataset.
